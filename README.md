# Chemiscope Website

B2B, procurement-friendly lead generation website for Chemiscope/Satol portfolio.

## Run locally

```bash
npm install
npm run dev
```

## Content editing

- Company data: `data/company.ts`
- Product catalog: `data/products.ts`
- Industries: `data/industries.ts`
- Solutions: `data/solutions.ts`

## Form handling

Inquiry submissions post to `POST /api/inquiry`.
